# xmaxify_website_repo

React + Node.js website foundation for Xmaxify.

## Project Structure

- `client/` - React frontend powered by Vite
- `server/` - Node.js Express API
- `package.json` - root workspace scripts

## Setup

```powershell
npm install
```

## Development

```powershell
npm run dev
```

Frontend runs on `http://localhost:5173`.
Backend runs on `http://localhost:5000`.

## Production Build

```powershell
npm run build
npm start
```
